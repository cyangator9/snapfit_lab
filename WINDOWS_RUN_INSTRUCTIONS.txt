QUICK START (3 STEPS):
----------------------

1. INSTALL ISAAC SIM + ISAAC LAB (One-time only)
   
   a) Download Omniverse Launcher:
      https://www.nvidia.com/en-us/omniverse/download/
   
   b) Install Isaac Sim from Omniverse Exchange tab
   
   c) Install Isaac Lab:
      - Open PowerShell as Administrator
      - Run:
        cd C:\
        git clone https://github.com/isaac-sim/IsaacLab.git
        cd IsaacLab
        .\isaaclab.bat --install

2. INSTALL THIS PROJECT (One-time only)
   
   - Open PowerShell
   - Run:
     cd C:\snapfit_lab
     C:\IsaacLab\isaaclab.bat -p -m pip install -e source\snapfit_lab

3. RUN TRAINING (Every time you want to train)
   
   METHOD A - EASIEST:
   -------------------
   1. Edit "run_training_windows.bat" in Notepad
   2. Check line 11: set ISAACLAB_PATH=C:\IsaacLab
      (change if your Isaac Lab is elsewhere)
   3. Save and close
   4. Double-click "run_training_windows.bat"
   
   METHOD B - POWERSHELL:
   ----------------------
   cd C:\snapfit_lab
   C:\IsaacLab\isaaclab.bat -p scripts\skrl\train.py --task=Template-Snapfit-Lab-Direct-v0 --num_envs=128 --headless
   
   METHOD C - WITH VISUALIZATION (slower but you see the robot!):
   --------------------------------------------------------------
   Double-click "run_training_with_gui.bat"

TROUBLESHOOTING:
----------------

PROBLEM: "Isaac Lab not found"
SOLUTION: Edit run_training_windows.bat and update ISAACLAB_PATH on line 11

PROBLEM: "Out of memory" or GPU crashes
SOLUTION: Reduce environments:
  - Edit run_training_windows.bat
  - Change "--num_envs=128" to "--num_envs=64" or lower

PROBLEM: Slow performance
SOLUTION: Make sure you're using headless mode (--headless flag)
  - Close other GPU applications (Chrome, Discord, games)
  - Use fewer environments

PROBLEM: Python/module errors
SOLUTION: Reinstall:
  cd C:\snapfit_lab
  C:\IsaacLab\isaaclab.bat -p -m pip install -e source\snapfit_lab --force-reinstall
